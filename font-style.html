FONT-STYLE{}

La propriété font-style permet de sélectionner une fonte italique (italic) ou oblique (oblique) parmi celles listées par font-family.
css
font-style: normal;
            font-style: italic;
            font-style: oblique;
            font-style: oblique 10deg;
            
            /* Valeurs globales */
            font-style: inherit;
            font-style: initial;
            font-style: revert;
            font-style: unset;
