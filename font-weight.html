FONT-WEIGHT

La propriété CSS font-weight permet de définir la graisse utilisée pour le texte. Les niveaux de graisse disponibles dépendent de la police (cf. font-family). Certaines fontes n'existent qu'avec les niveaux de graisses normal et bold.
css
/* Valeurs avec un mot-clé */
            font-weight: normal;
            font-weight: bold;
            
            /* Valeurs relatives à l'élément parent */
            font-weight: lighter;
            font-weight: bolder;
            
            /* Valeurs numériques */
            font-weight: 100;
            font-weight: 200;
            font-weight: 300;
            font-weight: 400; /* normal */
            font-weight: 500;
            font-weight: 600;
            font-weight: 700; /* gras */
            font-weight: 800;
            font-weight: 900;
            
            /* Valeurs globales */
            font-weight: inherit;
            font-weight: initial;
            font-weight: revert;
            font-weight: unset;
